Installation du certificat SSL

Vous devez installer le certificat SSL de la passerelle afin que votre navigateur puisse vérifier automatiquement l'intégrité de la connexion. Cela empêche quiconque de falsifier votre trafic et également protègera vos identifiants de connexion. Le certificat a été incorporé directement dans ce fichier HTML, et vous pouvez le télécharger içi:

Télécharger le certificat

Windows

Ces instructions fonctionnent pour Chrome et Internet Explorer. Firefox utilise son propre gestionnaire de certificats interne, mais est également facile à configurer.

  1. Téléchargez le certificat SSL intégré dans ce document.
  2. Double-cliquez pour ouvrir le certificat téléchargé.
  3. Cliquez sur Installer le certificat...
  4. L'Assistant d'importation de certificat démarrera. Cliquez sur Suivant.
  5. Sélectionnez Placer tous les certificats dans le magasin suivant puis cliquez Parcourir.
  6. Sélectionnez Autorités de certification racine de confiance puis cliquez OK.
  7. Cliquez Suivant en suite Finir.
  8. Confirmez que vous souhaitez installer le certificat en choisissant Oui.
  9. Redémarrez votre navigateur Web.
  10. Vous êtes prêt à vous connecter. Voir Connexion à votre passerelle Streisand.

macOS

Ces instructions fonctionnent pour Chrome et Safari. Firefox utilise son propre gestionnaire de certificats interne, mais est également facile à configurer.

  1. Lancez Trousseaux d'accès.
  2. Téléchargez le certificat SSL intégré dans ce document.
  3. Faites glisser le certificat téléchargé dans la section de session.
  4. Faites un clic-droit et choisissez Lire les informations.
  5. Cliquez sur le triangle à guache de Se fier pour ouvrir le menu contextuelle.
  6. Choisissez l'option Toujours approuver dans le menu deroulant pour le champ (SSL) Secure Sockets Layer.
  7. Fermer la fenêtre
  8. Une fenêtre apparaîtra qui vous demandera votre mot de passe; saissisez-le, puis cliquez sur Mettre à jour les réglages
  9. Vous êtes prêt à vous connecter. Voir Connexion à votre passerelle Streisand.

Androïde

Ces instructions fonctionnent pour Chrome Firefox pour Android utilise son propre gestionnaire de certificats interne mais ne fournit pas d'interface pour l'importation de certificats. Chrome est donc la manière recommandée de se connecter à la passerelle Streisand.

Moins sécurisé, mais facile

  1. Téléchargez le certificat SSL intégré dans ce document.
  2. Envoyez le certificat téléchargé par courrier électronique à un compte auquel vous pouvez accéder à partir de l'appareil Androïde.
  3. Ouvrez l'e-mail, puis appuyez sur la pièce jointe.
  4. Une invite apparaîtra, entez batterylow pour le Nom du certificat, et assurez-vous que VPN et applications est sélectionné pour la valeur de Utilisation du certificat.
  5. Tapez OK.
  6. Vous êtes prêt à vous connecter. Voir Connexion à votre passerelle Streisand.

Plus sécurisé

  1. Connectez l'appareil Androïde à votre ordinateur.
  2. Téléchargez le certificat SSL intégré dans ce document.
  3. Faites glisser le certificat téléchargé vers la racine du système de fichiers de l'appareil Androïde.
  4. Lancez l'application Paramètres.
  5. Faites défiler jusqu'à la section Appareil et tapez sur Sécurité.
  6. Faites défiler jusqu'à la section Stockage des identifiants et tapez Installer depuis la carte SD.
  7. Sélectionnez le certificat que vous avez copié sur votre téléphone.
  8. Entrez batterylow pour le Nom du certificat, et assurez-vous que VPN et applications est sélectionné pour la valeur de Utilisation du certificat.
  9. Tapez OK.
  10. Vous êtes prêt à vous connecter. Voir Connexion à votre passerelle Streisand.

iOS

Moins sécurisé, mais facile

  1. Téléchargez le certificat SSL intégré dans ce document.
  2. Envoyez le certificat téléchargé par courrier électronique à un compte auquel vous pouvez accéder à partir de l'appareil iOS.
  3. Ouvrez l'e-mail avec l'app Mail de iOS, en suite appuyez sur la pièce jointe.
  4. L'écran Installer le profil apparaîtra. Vous pouvez afficher les détails du certificat et vous assurer qu'ils correspondent aux informations de la section Verification SSL . Tapez Installer.
  5. Saisissez votre code NIP.
  6. Tapez Installer à nouveau lorsque l'avertissement apparaît.
  7. Tapez OK.
  8. Vous êtes prêt à vous connecter. Voir Connexion à votre passerelle Streisand.

Plus sécurisé

  1. Connectez le périphérique iOS à votre ordinateur macOS.
  2. Tapez Se fier si l'avertissement Faire confiance à cet ordinateur apparaît.
  3. Installez l'utilitaire Apple Configurator 2 depuis le Mac App Store.
  4. Lancez l'utilitaire Apple Configurator et acceptez la licence.
  5. Cliquez Start Preparing Devices.
  6. Cliquez Install Profiles....
  7. Allumez le téléphone et déverouille-le.
  8. Cliquez Next.
  9. Cliquez New... and entrez batterylow dans le champ Name dans la section General.
  10. Faites défiler jusqu'à la section Certificates.
  11. Téléchargez le certificat SSL intégré dans ce document.
  12. Cliquez Configure, sélectionnez le certificat téléchargé et cliquez sur Open.
  13. Cliquez Save.
  14. Cocher l'options à côté du profil batterylow que vous avez crée. Cliquez Next.
  15. L'écran Installer le profile apparaîtra sur votre appareil iOS. Tapez Installer.
  16. Tapez Installer à nouveau lorsque l'avertissement apparaît.
  17. Tapez OK.
  18. Vous êtes prêt à vous connecter. Voir Connexion à votre passerelle Streisand.

Chromium

  1. Cliquez sur le bouton Menu et accédez à Paramètres.
  2. Faites défiler jusqu'au bas de la fenêtre et cliquez sur Afficher les paramètres avancés...
  3. Faites défiler jusqu'au la section HTTPS/SSL et cliquez Gérer les certificats...
  4. Accedez a l'ognlet Autorités.
  5. Téléchargez le certificat SSL intégré dans ce document.
  6. Cliquez Importer..., sélectionnez le certificat, en suite cliquez Ouvrir.
  7. Une invite apparaîtra. Cochez l'option à côté de Confiez ce certificat pour identifier les sites Web et cliquez sur OK.
  8. Cliquez OK.
  9. Vous êtes prêt à vous connecter. Voir Connexion à votre passerelle Streisand.

Firefox

  1. Lancez Firefox.
  2. Ouvrez le panneau Options, et sélectionnez Préférences.
  3. Naviguez vers Vie privée et sécurité.
  4. Défiler jusqu'à la rubrique Sécurité > Certificats
  5. Cliquez sur Afficher les certificats.
  6. Une fenêtre apparaîtra. Cliquez sur l'onglet Autorités.
  7. Téléchargez le certificat SSL intégré dans ce document.
  8. Cliquez Importer..., sélectionnez le certificat, en suite cliquez Ouvrir.
  9. Une invite apparaîtra. Cochez l'option à côté de Confiez ce certificat pour identifier les sites Web et cliquez sur OK.
  10. Cliquez OK pour fermer le gestionaire des ceritificats, en suite fermer le panneau de Préférences.
  11. Vous êtes prêt à vous connecter. Voir Connexion à votre passerelle Streisand.

Verification SSL manuelle

L'option de vérification du certificat manuel est significativement moins sécurisée que l'installation du certificat en utilisant l'une des méthodes ci-dessus. Votre navigateur affichera un message d'avertissement et vous serai plus vulnérable aux attaques de l'homme du milieu car les empreintes digitales doivent être vérifiées lors de chaque tentative de connexion. Il devrait être utilisé avec beaucoup de soin, et seulement en dernier recours.

Les détails du certificat doivent correspondre aux informations suivantes:

Numéro de série

serial=FDE92E9057F9220F

Empreintes
SHA256 Fingerprint=06:8F:DB:D6:53:C5:AC:42:75:29:E7:4F:CC:3D:89:45:77:D5:77:DF:94:93:81:F0:A8:80:C1:FE:F5:72:CB:ED
SHA1 Fingerprint=33:82:63:48:61:94:91:B6:81:26:6A:64:01:E0:16:36:18:E8:76:E1
MD5 Fingerprint=2F:C1:54:78:03:EC:08:23:79:3F:61:D8:7F:E1:2E:5E

Si tout vérifie, vous êtes prêt à vous connecter. Voir Connexion à votre passerelle Streisand.
